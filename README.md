# Global Wakili - Legal Experience Platform (LXP)

## 🚀 Overview
**Global Wakili** is a next-generation legal management system tailored for the Kenyan market. It bridges the gap between law firm operations and client transparency.

## 🛠️ Key Modules
- **Global Accounts System:** Real-time Trust and Office account reconciliation.
- **Secure Client Portal:** Self-service dashboard for clients to track cases and pay invoices.
- **FinTech Integration:** Built-in M-Pesa STK Push for seamless legal fee collection.
- **Statutory Compliance:** KRA eTIMS ready with automated VAT-compliant invoicing.

## 🏗️ Technical Stack
- **Backend:** Node.js, Express
- **Database:** PostgreSQL via Prisma ORM
- **Security:** 2FA, AES-256 Encryption, and Role-Based Access Control (RBAC)

## 🚦 Getting Started
1. Clone the repo: `git clone https://github.com/mukigih-gif/global-wakili.git`
2. Install dependencies: `npm install`
3. Set up your `.env` file.
4. Run migrations: `npx prisma migrate dev`
5. Start the server: `node server.js`
